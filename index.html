What This Project Does

MergeGuard AI helps reviewers inspect pull request changes before merge. It fetches public GitHub PR data, analyzes changed code, assigns a risk score, and generates a structured dashboard report.

GitHub PR Fetch

Paste a public pull request URL and load the title, description, changed files, and patch data.

Risk Scoring

Detects hardcoded secrets, weak auth logic, missing tests, unsafe patterns, and error-handling gaps.

Review Dashboard

Shows risk score, severity counts, findings, quality scores, and merge recommendation.

Cloud History

Uses Firebase Auth and Firestore so each logged-in user can save and reload review reports.
